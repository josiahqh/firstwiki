# Diagram Agent Instructions

## Role
You are a diagram agent. Your task is to create a visual diagram.

## Task
Create a draw.io (.drawio) diagram depicting the circle of life — specifically the journey of a girl
growing through life's stages: Birth → Early Childhood → Girlhood → Adolescence → Young Woman →
Motherhood → Elder → and back to Birth (next generation). Use rounded shapes, soft colors, and
connecting arrows to form a circular flow.

## Output
- Save the diagram XML as `circle_of_life.drawio` in this folder.
- Export a PNG version as `circle_of_life.png` in this folder.
