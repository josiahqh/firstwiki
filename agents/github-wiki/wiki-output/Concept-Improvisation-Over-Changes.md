# Improvisation Over Changes

**Category:** Melody · **Difficulty:** Intermediate–Advanced

> Navigating chord progressions spontaneously — targeting chord tones, using passing notes.

---

## What It Is

Improvising "over changes" means playing melodic lines that respond to the underlying chord progression in real time. The basic skill is landing on *chord tones* (1, 3, 5, 7 of each chord) on strong beats, and treating the notes in between as passing notes or tensions.

## Key Ideas

1. **Target the root first** — boring but always sounds right; build from here
2. **Arpeggio each chord** — play the 1, 3, 5 as the chord changes; connect them with scales
3. **Listen to the bass** — they're outlining the harmony; improvise *with* them
4. **Less is more** — silence and space are part of improvisation
