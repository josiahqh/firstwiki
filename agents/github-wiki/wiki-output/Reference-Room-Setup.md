# Room Setup

Space configurations for different session types.

---

## Standard Configuration (Full Group)

```
                    [WALL]
         ┌─────────────────────────┐
         │  🖥️  Laptop + Interface  │
         │  🎛️  Mixer               │
         └────────────┬────────────┘
                      │ cables
    [Keys 🎹] ← left  │  right → [Acoustic guitar 🎸]
                      │
              [Drums / percussion 🥁]
                      │
              [Vocalist stands here 🎤]
```

- Vocalist faces away from monitor speakers to reduce feedback
- Drummer sits furthest from the laptop to minimise vibration transfer
- Leave a 1.5m corridor to the door — safety and cable access

## Small Group / Acoustic Session

- Gather in a circle — no PA needed for 2–3 people
- If recording, position the interface centrally and mic the room

## Cable Management

- All cables taped against the wall before playing
- XLR cables coiled overhand (never underhand — it kinks the cable)
- Leave one spare XLR and one spare ¼" jack accessible
