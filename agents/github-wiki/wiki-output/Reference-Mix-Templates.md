# Mix Templates

DAW template documentation — load at the start of each session to skip setup.

---

## Standard Jam Session Template

| Track # | Name | Input | Type | Key Plugins |
|---------|------|-------|------|-------------|
| 1 | Vocals | Interface 1 | Audio | Gate, EQ, light compression |
| 2 | Guitar | Interface 2 | Audio | EQ |
| 3–4 | Keys | Interface 3/4 | Audio Stereo | EQ |
| 5 | MIDI Keys | USB MIDI | Instrument | Software instrument |
| 6 | Click | Internal | Audio | Metronome |
| — | Master | — | Bus | Limiter (safety) |

## Headphone Mix Starting Points

| Player | Vocal | Guitar | Keys | Click |
|--------|-------|--------|------|-------|
| Vocals | 0 dB | -6 dB | -6 dB | -12 dB |
| Guitar | -6 dB | 0 dB | -6 dB | -6 dB |
| Keys | -6 dB | -6 dB | 0 dB | -6 dB |

Adjust to taste — these are just starting points.

---

**Session file naming:** `YYYY-MM-DD_Jam` — save a new file each session, never overwrite the template.
