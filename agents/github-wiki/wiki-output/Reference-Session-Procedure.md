# Session Procedure

Full step-by-step setup from walking in the door to making sound.

---

## Quick Start (Bare Minimum)

1. Power on the interface — wait for the light to stabilise
2. Open the DAW and confirm the interface is selected as audio device
3. Plug in a guitar or connect the MIDI keyboard
4. Set input gain — play/speak, watch the meter, stay in the green
5. Put on headphones or route to speakers
6. Play

---

## Full Setup — Power-On Order

Power everything on in this order to avoid speaker pops and phantom power accidents:

1. Power strip / mains (if needed)
2. **Audio interface** — wait for the light to stabilise
3. **Mixer** (if using)
4. **MIDI controller**
5. **Laptop** (or wake from sleep)
6. Open **DAW** — check audio device in preferences
7. **Monitors / speakers last**

## DAW Configuration

- Audio device: interface
- Sample rate: **48 kHz**
- Buffer: **128 samples** (monitoring) → **256** (tracking)

## Gain Staging

Play/sing at the loudest you'll be. Adjust input gain until peaks hit around **-12 dBFS**. Never clip — a clipped recording is unfixable.

## Routing Checklist

- [ ] Vocal mic → interface XLR input 1 → DAW track 1
- [ ] Guitar → interface jack input 2 → DAW track 2
- [ ] Keys → interface inputs 3/4 (stereo) → DAW track 3
- [ ] MIDI keyboard → USB → MIDI track
- [ ] Headphone mix set for each player
- [ ] Monitors at comfortable volume

## Power-Down Order

1. Turn down / mute monitors
2. Close DAW (save project if recording happened)
3. Mixer off → Interface off
4. Unplug and coil cables neatly
